Paradise Nursery - E-Plant Shopping Application
Project Description
Paradise Nursery is a comprehensive React-based web application designed for plant enthusiasts. The application provides a seamless shopping experience where users can browse through various categories of plants—such as Air Purifying, Aromatic, and Medicinal—view detailed descriptions, and manage their selections through a dynamic shopping cart.
